# Code of Conduct

## Our Commitment

We are committed to creating a welcoming, respectful, and inclusive environment for all contributors and participants in this project. Everyone is expected to contribute positively and professionally.

## Expected Behavior

- Be respectful and considerate in discussions.
- Use inclusive and welcoming language.
- Accept constructive feedback gracefully.
- Focus on improving the project and supporting the community.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Offensive, abusive, or inappropriate language.
- Trolling or intentionally disruptive behavior.
- Sharing private information without permission.

## Enforcement

Project maintainers have the right to remove, edit, or reject contributions that violate this Code of Conduct.

Repeated violations may result in temporary or permanent removal from the project.

## Scope

This Code of Conduct applies to:
- GitHub repository
- Issues and Pull Requests
- Discussions and project-related communications
